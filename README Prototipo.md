# 🎬 CINEMATRIX

Sistema web para catalogação e avaliação de filmes e séries, permitindo pesquisa de conteúdos, avaliações de usuários e organização de listas pessoais.

---

## 📖 Sobre o Projeto

Este projeto foi desenvolvido como uma iniciativa colaborativa com foco em estudo, prática de desenvolvimento full-stack e organização de dados audiovisuais.  
A plataforma permite que usuários consultem informações sobre filmes e séries, avaliem conteúdos e organizem suas próprias listas.
